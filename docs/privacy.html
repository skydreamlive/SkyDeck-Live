Politique de confidentialité

Dernière mise à jour : 15 juillet 2026.

Cette politique décrit le fonctionnement actuel de SkyDeck Live.

1. Données locales

SkyDeck Live peut enregistrer localement les noms, chemins et paramètres des logiciels configurés.

2. Connexion TikTok

Lorsqu’une fonction TikTok est activée, SkyDeck Live peut demander l’autorisation d’accéder aux informations de profil et statistiques autorisées par l’utilisateur.

3. Jetons d’accès

Les jetons d’accès doivent être conservés de manière sécurisée et ne doivent jamais être publiés sur GitHub.

4. Partage

SkyDeck Live ne vend pas les données personnelles.

5. Contact

Les demandes peuvent être envoyées via GitHub Issues.
