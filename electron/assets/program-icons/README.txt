Icônes personnalisées SkyDeck
================================

Dépose ici les images personnalisées des logiciels.

Le nom du fichier doit correspondre exactement à l'identifiant
du logiciel dans programs.json.

Exemples :

tikfinity.png
aplv2.png
logitech.png
navigraph-hub.png
msfs24.png

Formats acceptés :
- PNG
- JPG / JPEG
- WEBP

Ordre utilisé par SkyDeck :
1. Icône personnalisée présente dans ce dossier.
2. Icône Windows extraite du fichier EXE.
3. Initiales du logiciel.

Taille recommandée :
256 × 256 pixels, fond transparent si possible.
